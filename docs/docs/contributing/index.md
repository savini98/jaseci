## How to Contribute

1. **Fork the repository**: [github.com/Jaseci-Labs/jaseci](https://github.com/Jaseci-Labs/jaseci)
2. **Set up development environment**: Follow the [Contributing Guide](../internals/contrib.md)
3. **Find an issue**: Check the issue tracker for good first issues
4. **Submit a PR**: Follow the contribution guidelines
