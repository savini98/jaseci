# Jac scale roadmap

`jac start --scale` not only simplifies application deployment but also supports advanced configurations.

### Upcoming features

1. Hot reload functionality
2. Web socket support
3. SSO support for
    - Apple
    - Meta
    - GIthub
4. Schedule walkers to run at defined time
5. Auto creation of Dockerfile
6. Support for horizontal scaling for k8s
7. Dashboard to check status of deployment
8. CORS support
